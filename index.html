About the Volcano Game

Made for Game Off 2018, but has been an idea brewing for awhile now.

The Volcano game is a mixture of different ideas and is influenced by a variety of different games. The decision-based gameplay, and having only 1 life, is influenced by Rogue. Unlike Rogue, the interface is entirely different. I wanted this game to be more an experiment in map generation and combat mechanics, rather than graphics.
