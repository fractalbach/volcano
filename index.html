About the Volcano Game

Made for Game Off 2018, but has been an idea brewing for awhile now.

The Volcano Game is a combination of a text-based adventure game like ADVENT, with single-life and turn-based combat like Rouge, and is built as a web page. The unique feature is the procedurally generated Graph-based game world.

I wanted to create a game that was built around a non-linear world. You have to piece together it‚Äôs structure as you go. Using a Graph allows for freedom in the creation of interesting structures. Paths from start to finish allow players to complete the level, and Cycles can cause you to end up back in the same place again.

The idea of a ‚Äúvolcano‚Äù came from these random graphs. Nodes are like rooms or chambers, and Edges are like tunnels and passageways. Without a compass, it would be easy to become lost. This aspect of the game come from ADVENT, but I didn‚Äôt want players to draw their own map: I think it‚Äôs fun to see the map as you explore it.

It‚Äôs not just a maze. When you walk into a room, there‚Äôs something you have to solve before you unlock its secrets and gain access to its other passageways. I added a simple combat system so that the player had to ‚Äúsurvive‚Äù their way through the volcano. Currently, the game can be thought of as a Shortest path problem, because there is a cost to solve room. You only have a finite amount of health, and each room will a take its toll.
